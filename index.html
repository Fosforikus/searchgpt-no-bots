🧪 TEST A3 – Blocage total des bots

🔍 Objectif du test

Ce test vise à déterminer si SearchGPT peut accéder à une page qui n’est indexée par **aucun moteur de recherche**. Le fichier robots.txt interdit tous les bots. Le contenu contient une phrase absurde unique, totalement inédite sur le web.

🧾 Phrase absurde à détecter

Le trombone antigravité de Maubeuge récite les tables de multiplication en sanskrit fluorescent pendant les éclipses musicales.

📁 robots.txt appliqué

User-agent: *
Disallow: /

📌 Résultat attendu

Si cette phrase est détectée par SearchGPT, cela signifie que le modèle accède au contenu autrement que via Google ou Bing.
